Load this dataset directly with the Datasets library
First, install dependency and clone SEACrowd 
pip install datasets
git clone https://github.com/SEACrowd/sea-crowd.git
cd sea-crowd

Then, the dataset can be downloaded locally by the python script below: 
from datasets import load_dataset

path = "sea-crowd/seacrowd/seacrowd_datasets/TMP"
dataset = load_dataset(path)

# see dataset sample:
print(dataset['train'].to_pandas())
